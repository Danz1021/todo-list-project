## 使用方法

### 基本操作
1. **新增任務**: 在輸入框中輸入任務內容，選擇優先級後點擊「新增」
2. **完成任務**: 勾選任務前的核取方塊
3. **刪除任務**: 點擊任務右側的「×」按鈕
4. **篩選任務**: 使用底部的篩選按鈕檢視不同狀態或優先級的任務

### 進階功能
- **鍵盤快捷鍵**: 在輸入框中按 Enter 快速新增任務
- **進度慶祝**: 達成 25%、50%、75%、100% 完成度時觀看動畫
- **每日重置**: 應用程式會在隔日自動清空前一天的任務
